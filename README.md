# OrionBC

BMI212 project

### OrionBC environment requirements
**Python: 3.12**

**Environment Setup Example:**

> conda create -n orionbc python=3.12

> conda activate orionbc

> pip install -r requirements.txt

## run.ipynb

### 1) Training + testing the model on breast cancer tissue miRNA count samples (Cancer Vs. Control)
### 2) Training + testing the model on breast cancer circulating miRNA count samples (Cancer Vs. Control)
### 3) Training the model on tissue samples and testing on circulating samples
### 4) Training the model on large (n=16k) pan-cancer dataset of serum miRNA samples (Binary)
### 5) Training the model on pan-cancer dataset (Cancer type classification)
